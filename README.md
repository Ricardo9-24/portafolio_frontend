# Mi Portafolio Personal - React & TypeScript

¡Bienvenido a mi portafolio personal! Este proyecto fue creado para mostrar mis habilidades como desarrollador frontend autodidacta con React y TypeScript. Aquí encontrarás una demostración de mi capacidad para construir interfaces de usuario interactivas y funcionales, utilizando conceptos clave y herramientas modernas.

## Descripción General

Este portafolio es una vitrina de mis proyectos y habilidades. Fue desarrollado con un enfoque en la reutilización de componentes, la gestión eficiente del estado y la creación de una experiencia de usuario atractiva y accesible.

## Tecnologías Utilizadas

*   **React:** Librería de JavaScript para construir interfaces de usuario.
*   **TypeScript:** Superset de JavaScript que añade tipado estático.
*   **CSS & Bootstrap:** Estilos personalizados con CSS y la flexibilidad de Bootstrap.
*   **EmailJS:** Librería para el envío de correos electrónicos desde el frontend.
*   **lucide-react:** Librería de iconos vectoriales.
*   **framer-motion:** Librería para crear animaciones fluidas y atractivas.

## Habilidades Demostradas

En este proyecto, he aplicado mis conocimientos como desarrollador autodidacta en React, demostrando un nivel junior con las siguientes habilidades:

*   **Componentes y Componentes Reutilizables:**
    *   Creación de componentes modulares y reutilizables para una estructura de código limpia y mantenible.
    *   Ejemplos: Componentes de tarjetas de proyecto, encabezados y pies de página reutilizables.
*   **Hooks (useState y useEffect):**
    *   Gestión del estado de la aplicación con `useState` para crear interfaces dinámicas.
    *   Utilización de `useEffect` para manejar efectos secundarios como la carga de datos inicial y la suscripción a eventos.
*   **Formularios Manejados con Estados:**
    *   Implementación de formularios controlados donde el estado de los inputs se gestiona con React, permitiendo la validación y el control de los datos.
*   **Estilos con CSS Crudo y Bootstrap:**
    *   Uso de CSS personalizado para dar estilo a los componentes y adaptar el diseño a mis preferencias.
    *   Aprovechamiento de Bootstrap para una maquetación rápida y responsiva.
*   **Manejo de Expresiones y Listado de Datos desde JSON:**
    *   Renderizado dinámico de contenido a partir de datos almacenados en archivos JSON.
    *   Uso de expresiones JSX para mostrar información de manera eficiente.
*   **Envío de Correo con EmailJS:**
    *   Integración de EmailJS para permitir a los visitantes contactarme directamente desde el portafolio.
*   **Tema Claro/Oscuro con LocalStorage:**
    *   Implementación de un tema claro/oscuro persistente utilizando `localStorage` para recordar la preferencia del usuario.
*   **Iconos con lucide-react:**
    *   Utilización de iconos vectoriales de lucide-react para mejorar la estética y la usabilidad del portafolio.
*   **Animaciones con framer-motion:**
    *   Adición de animaciones sutiles y atractivas con framer-motion para mejorar la experiencia del usuario.
*   **Modales con CSS, useState y Renderizado Condicional:**
    *   Creación de modales personalizados utilizando CSS para el diseño, `useState` para controlar la visibilidad y renderizado condicional para mostrar el modal solo cuando es necesario.
